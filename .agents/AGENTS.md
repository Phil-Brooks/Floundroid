# Custom Rules for Antigravity

To minimize credit usage and optimize efficiency, please follow these behavioral constraints at all times:

## Response Optimization
- **Keep responses concise**: Provide brief, direct answers. Avoid verbose explanations or summarizing code unless specifically requested.
- **No redundant summaries**: Do not summarize the contents of implementation plans, walkthroughs, or files that are already visible in artifacts or links.

## Tool and File Access
- **Targeted file reading**: When viewing files, read only the relevant sections (using specific line ranges) rather than reading the entire file. Limit file viewing to under 200 lines at a time.
- **Narrow search scope**: Use precise query terms and filter search paths to specific subdirectories to avoid large search results.

## Agent Operations
- **Minimize subagents**: Do not spawn subagents unless explicitly requested by the user or required for complex parallel tasks.
- **Avoid over-planning**: For straightforward modifications or simple bug fixes, do not create an implementation plan unless requested or the task is highly ambiguous.
